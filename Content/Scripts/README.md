# Gameplay Scripts

Scripts created from the editor's **Create Script** button are stored here.
They are registered automatically in the shared game module and are compiled for
both Editor Play and standalone player builds.

Double-click a script in the Assets panel to edit it. Use **Compile Scripts &
Restart** to build native changes into both targets. The Script Editor can also open
files in VS Code, Visual Studio, Rider, or a configured custom editor.
